PROJECT:
IN COLD BLOOD, 2017 SHORTLIST COVER DESIGN.

Skills used; Illustrator, InDesign, Poster Design, UI.

About The Project

Orderly is a Stock management software design from the ground up for Starbucks in Europe, Asia and Africa, allowing this branches not only the ability to save millions of pound but also reduce waste. This software is now going to made available as a subscription service, allowing other companies to the user it in the same way. However, the software needed a redesign to allow it to be used in this manner. As the joiner design, I was tasked to create a collection of icons to give the system a more fished look and bring element together.

This somewhat small project became bigger and the list became longer, leading to the creation of 600 + icons; ranging from home, nav, setting to boxes, joint trading and theoretical over reality. All icons come in Bold, Regular and Inverted to allow for any placement. Because of the large number of icons the developer needed to be able to look up the icon with its name and so a poster was created to showcase the icons and give needed information.
